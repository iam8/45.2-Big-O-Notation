# Step One: Simplifying Expressions

Ioana A Mititean

1. O(n + 10)
    - Simplified: O(n)
1. O(100 * n)
    - Simplified: O(n)
1. O(25)
    - Simplified: O(1)
1. O(n^2 + n^3)
    - Simplified: O(n^3)
1. O(n + n + n + n)
    - Simplified: O(n)
1. O(1000 * log(n) + n)
    - Simplified: O(n)
1. O(1000 * n * log(n) + n)
    - Simplified: O(n log(n))
1. O(2^n + n^2)
    - Simplified: O(2^n)
1. O(5 + 3 + 1)
    - Simplified: O(1)
1. O(n + n^(1/2) + n^2 + n * log(n)^10)
    - Simplified: O(n log(n)^10)
